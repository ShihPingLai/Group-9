## 單一正電荷在空間中的磁場分佈 ##

### 目的 
=======
透過使用者輸入不同位置，以python模擬在空間中單一正電荷周圍的2D或3D場分佈圖。


### 使用說明
======
> 使用者需輸入空間分佈


### 結果
======

在過程中我們發現，3D圖較能顯示出真實電場方向分佈，但2D圖較能顯示出真實電場的大小分布
